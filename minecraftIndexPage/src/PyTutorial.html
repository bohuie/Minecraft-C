How to Install Python

Introduction:

Python is a widely used high-level programming language. Since Python has been created, it has been gaining popularity and is considered as one of the most popular and flexible server-side programming languages. Unlike most Linux distributions, Windows does not come with the Python programming language by default. However, you can install Python on your Windows server or local machine in just a few easy steps

Download Package:

Get your executable package here.
You can also find a specific release from the link above

Run Executable Installer:

Run the Python installer. Then follow the instructions.

Verify Python Was Installed On Windows:

1. Navigate to the directory in which Python was installed on the system.
2. Double-click python.exe.
3. You should be able to see a window that is similar to this.

If you can see the version of your Python, then you are good to code now!
Enjoy coding~
